JE org_id Organization Identification numeric  
JE je_id Journal entry identification numeric A serial number assigned by the system not visible to the user.
JE je_voucher Journal entry voucher number numeric A serial number assigned by the system, from a starting point specified by the user. Voucher sequence does not suffer any hole in it. Each organization has its own voucher sequence. The accountant transcripts the voucher number on the source document. Every single voucher has to be accounted for. The voucher serves as the audit trail.
