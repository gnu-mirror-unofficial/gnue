GNU Enterprise

This project has been decommissioned. This web page is kept here for historical purposes only.

GNU Enterprise is a project aimed at developing a complete enterprise level business environment. Currently, the scope of GNU Enterprise is limited to a set of tools to develop interactive database applications.

Downloading GNU Enterprise

GNU Enterprise can be found on the main GNU ftp server via HTTPS,  HTTP or FTP.

Documentation

Documentation for GNU Enterprise is included in the source distribution. You may also find more information about GNU Enterprise by looking at /usr/share/doc/gnue/, or similar directories on your system.

Getting involved

Development of GNU Enterprise, and GNU in general, is a volunteer effort, and you can contribute. For information, please read How to help GNU. If you'd like to get involved, it's a good idea to join the discussion mailing list (see above).

Development
For development sources and other information, please see the GNU Enterprise project page at savannah.gnu.org.
