Requests generated internally
=============================


ClipboardGet
------------

  Description:
     This event signals to the current entry's input handler to
     enter edit mode.

  Input Parameters: n/a

  Return Parameters: n/a

  Returns:
     The current text of the clipboard, or None if clipboard is empty.

  Comments: n/a



ClipboardSet
------------

  Description:
     This event requests (from the UI) that the clipboard contents
     be replaced.

  Input Parameters:
     text         The text to place on the clipboard

  Return Parameters: n/a

  Returns: n/a

  Comments: n/a



BeginEditMode
-------------

  Description:
     This event signals to the current entry's input handler to
     enter edit mode.

  Input Parameters: n/a

  Return Parameters: n/a

  Returns: n/a

  Comments: n/a



EndEditMode
-----------

  Description:
     This event signals to the current entry's input handler to
     leave edit mode.  If event._error__ is set, then the input
     was not valid and, therefore, the entry is still in edit
     mode.

  Input Parameters: n/a

  Return Parameters:
     modified     If set to true, then the field was modified
                  sometime after the beginEditMode event.

  Returns:
     Final value of field

  Comments: n/a
